# sql-copilot
Meu aprendizado sobre Introdução de consultas com I.A.


Seguindo a trilha de aprendizado sobre inteligência artificial, sua variedade de prompt's e a usabilidade na tecnologia, através do Microsoft Copilot estruturei um banco de dados sobre um sistema de cartas. Meu intuito principal ao expôr este repositório é demonstrar a facilidade que as I.A's transmitem no desenvolvimento e a sua utilidade se usada legitimamente.
Toda a resposta da I.A depende da sua capacidade de trazer informações e utilizá-la como ferramenta de análise, e não para gerar os dados.
